Controls

  • Press Key on keyboard to move Snake Up.

  • Press Key on keyboard to move Snake Down.

  • Press Key on keyboard to move Snake Right.

  • Press Key on keyboard to move Snake Left.

  • Press Key on keyboard to Pause the Game and return to Main Menu.

Ojective of The Snake Game

  • Feed the Hungry Snake by moving it across the field.
  • Snake will pass the boundary and will enter from opposite side.
  • You will lose the Game if the Snake eat itself.

Game Over! Press any key to start a new game.
